= Summary

adding summary can be done in 3 ways:

  • Leave at it is, the summarization will be generated automatically albeit not customizable
  • Add “< !–more– >” section in markdown file to indicate the end of paragraph which will be truncated (remove the spaces)
  • Manually compose a summary within the commented frontmatter variable summary
